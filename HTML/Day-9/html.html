What is HTML?

  1. HTML stands for Hyper Text Markup Language
  2. HTML is the standard markup language for creating Web pages
  3. HTML describes the structure of a Web page
  4. HTML consists of a series of elements
  5. HTML elements tell the browser how to display the content
  6. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is Blocl level tags?

Block-level elements in HTML always start on a new line and occupy the full width available on the webpage. They are mainly used to create the structure of a webpage. Examples of block-level tags are <div>, <p>, <h1>, <section>, and <table>.

What is Inline level?

On the other hand, inline elements do not start on a new line and only take up the space required for their content. They are generally used for styling or displaying small parts of content within a line. Examples of inline tags are <span>, <a>, <b>, <i>, and <code>.while inline elements appear within the same line of content.

What is Semetric Tag?

Semantic tags in HTML are tags that clearly describe the meaning and purpose of the content they contain. They help both browsers and developers understand the structure of a webpage easily. Semantic tags improve readability, accessibility, and SEO (Search Engine Optimization). Examples of semantic tags include <header> for the top section of a webpage, <nav> for navigation links, <section> for grouping related content, <article> for independent content, <aside> for side content, and <footer> for the bottom section of a webpage. Unlike non-semantic tags such as <div> and <span>, semantic tags provide meaningful information about the content inside them.

What is Non-Semetric Tag?

Non-semantic tags in HTML are tags that do not describe the meaning or purpose of the content they contain. They are mainly used for styling or grouping content without giving any information about the structure of the webpage. The most common non-semantic tags are <div> and <span>. The <div> tag is used as a block-level container, while the <span> tag is used as an inline container. Unlike semantic tags such as <header>, <article>, and <footer>, non-semantic tags do not tell browsers or developers what type of content is inside them.

What is Attribute?

HTML attributes are special properties used inside HTML tags to provide additional information about an element. Attributes help control the behavior, appearance, or functionality of elements on a webpage. They are always written inside the opening tag and usually come in a name and value pair format, such as attribute="value".

Attributes are very important because they allow developers to customize elements. Different tags use different attributes depending on their purpose. For instance, the <img> tag uses the src attribute to define the image source and the alt attribute to provide alternative text.

Some commonly used HTML attributes are:

Attribute Description
id Gives a unique identity to an element
class Groups elements for CSS or JavaScript
href Specifies a hyperlink destination
src Specifies image or media source
alt Provides alternative text
style Adds inline CSS styling
placeholder Displays hint text in input fields
required Makes a form field mandatory

Overall, HTML attributes make webpages more interactive, meaningful, and functional by giving extra instructions to HTML elements.

What is Table?

An HTML table is used to organize and display data in rows and columns, similar to a spreadsheet. Tables are created using the <table> tag. Inside the table, rows are created using the <tr> tag, table headings are defined using <th>, and table data cells are added using <td>. HTML tables are commonly used to display structured information such as student details, employee records, product lists, and timetables.

What is Video?

The <video> tag in HTML is used to display videos on a webpage. It allows users to play, pause, and control video content directly in the browser without using external plugins. HTML5 introduced the <video> element to make multimedia integration easier and more efficient.

  1. <video> creates the video player.
  2. <source> specifies the video file and its format.
  3. src defines the video file location.
  4. type tells the browser the video format.
  5. controls adds play, pause, volume, and fullscreen buttons.

What is Audio?

The <audio> tag in HTML is used to add sound or music to a webpage. It allows users to play audio files directly in the browser without using external plugins. HTML5 introduced the <audio> element to make audio integration simple and efficient.

  1. <audio> creates the audio player.
  2. <source> specifies the video file and its format.
  3. src defines the video file location.
  4. type tells the browser the video format.
  5. controls adds play, pause, volume, and fullscreen buttons.

What is iframe?

The <iframe> tag in HTML is used to embed another webpage, video, map, or external content inside the current webpage. The word iframe stands for “Inline Frame.” It allows one HTML document to be displayed within another HTML document.

Common attributes used with the <iframe> tag:

Attribute purpose
src Specifies the webpage URL
width Sets iframe width
height Sets iframe height
allowfullscreen Allows fullscreen mode

The <iframe> tag is commonly used to embed:

  1. YouTube videos
  2. Google Maps
  3. Other websites
  4. Online documents or forms

What is Fav icon?

A favicon is a small icon displayed in the browser tab beside the webpage title. It helps identify a website easily and improves the website’s appearance and branding. Favicons are usually added using the <link> tag inside the <head> section of an HTML document.

Favicons appear in:

  1. Browser tabs
  2. Bookmarks
  3. Browser history
  4. Mobile shortcuts

What is Meta tag?

Meta tags in HTML are special tags used inside the <head> section of a webpage to provide information about the webpage to browsers and search engines. They do not display content on the webpage but help control webpage behavior, improve SEO (Search Engine Optimization), and define page settings.

Commonly used meta tags:

  • charset - Defines character encoding
  • viewport - Makes webpage responsive on devices
  • description - Gives webpage description for search engines
  • keywords - Specifies related keywords
  • author - Defines webpage author
