1) What is a Variable? A variable is a named container used to store data values in a program. Example: let name = "Abinaya"; here name is a container. 2. What is Declaration? Declaration means creating a variable by specifying its name. Example: let age; Here, age is declared but not assigned a value. 3. What is Initialization? Initialization means assigning a value to a variable. Example: let age = 21; Here, age is declared and initialized with 21. 4.
