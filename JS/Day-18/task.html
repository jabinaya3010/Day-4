HTML, DOM, React, Components & Functions

1. What is DOM?

DOM stands for Document Object Model. It is a programming interface that represents an HTML document as a tree of objects. JavaScript uses the DOM to access and modify HTML elements, CSS styles, and page content dynamically.

Uses of DOM

  • Change HTML content
  • Change CSS styles
  • Add or remove elements
  • Handle user events

2. What is React?

React is a JavaScript library used to build fast, interactive and reusable User Interfaces (UI). It is mainly used for creating Single Page Applications (SPA).

Features

  • Reusable Components
  • Virtual DOM
  • Fast Performance
  • Easy State Management
  • One-way Data Binding
function App() { return ( <h1>Hello React!</h1> ); }

3. What are Components?

A Component is a reusable piece of UI in React. Each component contains its own HTML, CSS and JavaScript logic. Components help make code modular and reusable.

Types of Components

  1. Functional Component
  2. Class Component
function Welcome() { return <h2>Welcome to React</h2>; }

4. What are Functions?

A Function is a block of reusable code that performs a specific task. It executes only when it is called. Functions help reduce code duplication and improve readability.

Example

function greet() { alert("Hello World"); } greet();
